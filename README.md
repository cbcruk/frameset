# Frameset

여러 웹사이트를 한 화면에 분할하여 동시에 볼 수 있는 Electron 기반 데스크톱 앱입니다.

## Features

- **다중 웹사이트 분할 뷰**: 1~4개의 웹사이트를 동시에 표시
- **모바일 뷰 모드**: 모바일 User-Agent로 사이트 로드
- **광고 차단**: @ghostery/adblocker-electron 기반 광고 차단
- **커스텀 필터**: uBlock Origin/AdBlock Plus 형식 지원
- **사이트 순서 정렬**: 드래그 없이 버튼으로 순서 변경
- **X-Frame-Options 우회**: iframe 제한 사이트도 로드 가능

## Usage

1. 사이드바에서 URL 입력 후 추가
2. 분할 버튼(1~4)으로 화면 분할 개수 설정
3. 모바일 뷰 토글로 모바일/데스크톱 전환
4. 각 셀의 버튼으로 새로고침, 뒤로/앞으로, 개발자 도구 사용
